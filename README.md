# Display Product Names Using JavaScript Array

## Description  
This project creates an array containing at least ten product names and dynamically displays them as a list on a webpage using JavaScript. It demonstrates how to manipulate the DOM to show array data to users with the help of HTML and CSS for structure and styling.

## Features  
- An array with no less than 10 product names.  
- Dynamically generates and displays the product list on the page.  
- Simple and clean presentation using HTML and CSS to highlight JavaScript functionality.

## Technologies Used  
- HTML5  
- CSS3  
- JavaScript (Array & DOM Manipulation)

## Purpose  
The goal of this task is to practice creating and using arrays in JavaScript and dynamically displaying their content on a styled webpage by manipulating the DOM.
